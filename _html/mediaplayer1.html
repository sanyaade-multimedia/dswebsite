The mediaDROID-90 is an inexpensive player designed to deliver dynamic animated content, including HD videos. It is powered by the full version of our SignagePlayer app, so you get all the same set of features in a low powered, plug-N-play, stateless, Android based device. The device is about the size of a matchbox which makes it ideal for small space installations. It ships with a special baked Android operating system, that was developed by MediaSignage to deliver a none interrupted continues playback of your campaigns. Simply plug the enclosed HDMI port onto a screen and use the built in WIFI module to connect to your local wireless router (or via an external Ethernet dongle - sold separately). The device can be powered using a screen’s USB port, therefore an external power supply is not required. The mediaDROID-90 can cache up to 4GB of campaign resources on its internal Flash memory, so it can operate even when the connection to the internet is lost with no interruption to your audience.

Choose the mediaDROID-90h to drive any screen in Horizontal / Landscape mode or mediaDROID-90v that will drive any screen in Vertical / Portrait mode; both are compatible with any HDMI screen.
